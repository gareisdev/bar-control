# BarControl

Sistema de gestion de pedidos para bares.

## 🛠️ Tecnologías
Backend: Go
Framework API: Echo
Base de Datos: PostgreSQL
ORM: sqlx

## Iniciar el proyecto

1. Instala las dependencias

```shell
go mod tidy
```

2. Ejecuta el proyecto
``` shell
make run
```
